# Comandos para Terraform.

## Iniciar.
    terraform init

## Dar formato.
    terraform fmt

## Ver el state.
    terraform show

## Vemos todos los cambios por aplicar.
    terraform plan

## Aplicamos los cambios.
    terraform apply

## Eliminamos todo.
    terraform destroy
